# RazerSteelseriesWave
a C implementation of my wave effect

# Building
```shell
mkdir build
cd build
cmake ..
make
```


## Dependencies 
- DBus
- [RazerInterface](https://github.com/Coolio4691/RazerInterface)
- HIDAPI
